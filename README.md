# Job Agent

Локальный агент подбора вакансий. Ночью собирает вакансии из Telegram-каналов и
агрегаторов, скорит под резюме пользователя его же AI (bring-your-own), отдаёт
`.xlsx` и/или дайджест в личный Telegram-бот. Всё работает на машине пользователя,
данные никуда не уходят, кроме как к выбранному AI-движку и web-поиску.

## Статус

Проект в разработке. Полный план реализации, инварианты и порядок задач —
[`docs/plans/job-agent-plan.md`](docs/plans/job-agent-plan.md). Конвенции проекта —

## Разработка

Требуется [`uv`](https://docs.astral.sh/uv/).

```sh
uv sync                       # установить зависимости
uv run ruff check src tests   # линт
uv run pytest -q              # тесты
```
