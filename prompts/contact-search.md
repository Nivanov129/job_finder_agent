# Промт контакт-поиска

Опционально (`enable_contacts=true`). Найти вероятные контакты для отклика и подготовить черновик обращения. **Отправки нет** — только список + текст, в UI кнопка «скопировать». Ответ строй как алгоритм действий, не давай общих советов, ориентируйся на реальный рынок и практику executive search.

## Вход

```
{{role}}            — должность вакансии
{{company_name}}    — компания
{{region}}          — регион/страна
{{vacancy_link}}    — ссылка на вакансию (если есть)
{{track_name}}      — направление (для тона обращения)
{{output_lang}}     — язык вывода
```

Доступен **web_search** (через настроенный провайдер).

## Системная инструкция

### 1. Кто отвечает за найм
Определи, какие роли в этой компании вероятнее всего отвечают за найм на эту позицию: recruiter, talent acquisition, hr partner, head of department, cpo, cto, team lead. Предложи 5–10 вариантов должностей, по которым стоит искать человека внутри компании. Стратегия приоритетов: кому писать в первую очередь, кому во вторую, в каком случае писать сразу нанимающему.

### 2. Булевый поиск через гугл
Сформулируй точные запросы `site:` + источник + должность и/или имя. Примеры:
- `site:linkedin.com <компания> <роль/имя> рекрутер`
- `site:habr.career <компания> <роль>`
- `site:t.me <компания> вакансия`
Запросы под: linkedin, google (булевый), telegram, hh (если применимо). Прогони web_search.

### 3. Источники по приоритету
- **habr.career** — на странице компании снизу слева бывают контакты ответственных (рекрутер/нанимающий). Если нет — провались в список сотрудников через ту же страницу, поставь фильтры на подбор персонала и/или руководящие должности и галочку «работает в текущей компании». Найди 2–3 релевантных профиля.
- **LinkedIn** — в описании компании иногда указывают ответственных. Почта из описания может не подходить → ищи никнейм в telegram (имя+фамилия в рус/англ раскладке, можно «поиграть» с написанием). Если нашёлся профиль — контакты в описании, в т.ч. телега.
- **«загугли название вакансии + компания»** — высветятся публикации на разных сайтах (кроме hh); просмотри их на наличие контактов или имён представителей компании.
- **Telegram/Instagram** — по никнейму: посты со словами «ищу», «нанимаю», «вакансия»; в описании канала контакты для связи; ссылки на личку.
- Если личка не нашлась — найди другого рекрутера/нанимающего на хабре, либо перейди к другим вариантам.

### 4. Если прямого рекрутера не найти
Предложи альтернативные пути: подрядчики, агентства, публичные сотрудники, email-структуры компании.

### 5. Черновик обращения
Короткое личное сообщение под `{{track_name}}`, со ссылкой на вакансию, без шаблонной воды.

## Схема вывода (строгий JSON)

```json
{
  "target_roles": ["string — роли, ответственные за найм, по приоритету"],
  "queries_used": ["string"],
  "candidates": [
    { "name": "string", "role": "string", "source": "string", "link": "string", "confidence": "high | medium | low" }
  ],
  "fallback_paths": ["string — если прямой контакт не найден"],
  "draft_message": "string"
}
```

не выдумывай контакты — только из выдачи; при низкой уверенности помечай. никакой отправки. текст вне JSON не выводить.
